Sobre esta API

Esta é uma API RESTful construída com Spring Boot, projetada para servir os dados do meu portfólio profissional. Ela fornece informações sobre meu perfil, habilidades e projetos de forma estruturada (JSON).

Endpoints Disponíveis

Para interagir com a API, utilize os seguintes endpoints:

GET /profile

Retorna um objeto JSON com informações do meu perfil profissional, incluindo links para LinkedIn e GitHub.

GET /skills

Retorna uma lista das minhas habilidades técnicas, classificadas por categoria e nível de proficiência.

GET /projects

Retorna uma lista dos principais projetos em que trabalhei, com descrições, status e tecnologias utilizadas.
